IPL Cricket Analytics Engine

A comprehensive cricket analytics project built using Python, Pandas, Matplotlib, and Streamlit.

Features

Batting Analytics

* Top Run Scorers
* Strike Rate Analysis
* Batting Average
* Boundary Percentage

Bowling Analytics

* Top Wicket Takers
* Economy Rate

Team Analytics

* Team Win Percentage
* Toss Impact Analysis

Venue Analytics

* Highest Scoring Venues
* Venue-wise Toss Impact

Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Git & GitHub

Dataset

IPL Ball-by-Ball Dataset (2008–2025)

Project Structure

data → analytics → visualization → dashboard
